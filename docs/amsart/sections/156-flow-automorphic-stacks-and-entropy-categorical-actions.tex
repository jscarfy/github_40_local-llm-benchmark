\section{Flow Automorphic Stacks and Entropy Categorical Actions}
\subsection{Flow automorphic stacks}
We introduce flow automorphic stacks as deformation-aware moduli objects carrying entropy stratifications and
flow-equivariant structural maps encoding automorphic/arithmetic behavior.

\subsection{Entropy categorical actions}
We define entropy categorical actions acting on sheaf-like or module-like data over flow automorphic stacks,
tracking arithmetic invariants through categorical trace constructions.

\begin{highlightedSyntaxPhenomenon}[Automorphic geometry as entropy-categorical action]
Automorphic structure is expressed as an \emph{entropy-categorical action} on flow stacks, turning arithmetic
invariants into categorical traces stable under deformation.
\end{highlightedSyntaxPhenomenon}
